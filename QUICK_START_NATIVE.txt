## INSTALLATION (3 COMMANDS)

```powershell
# 1. Clean install
Remove-Item -Recurse -Force node_modules -ErrorAction SilentlyContinue
Remove-Item -Force package-lock.json -ErrorAction SilentlyContinue
npm install --legacy-peer-deps

# 2. Build and run
npm run build
npx cap sync android
npx cap run android
```

## HOW IT WORKS NOW

### For Users:
1. Open app → Settings → Notification Sound
2. Guide modal appears with instructions
3. Tap "Open Settings" button
4. **Android settings open directly** ✨
5. Tap "Sound" and choose any sound
6. Done! Sound changed

### Simple & Professional
- Uses standard Android notification settings
- No custom file picker
- No audio trimming
- No bugs
- Works everywhere
