🛡️ VALIDATION LOGIC
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

When user picks a file:

1. Check MIME type: Does it start with "audio/"?
2. Check extension: Is it .mp3, .wav, .m4a, etc.?

If EITHER passes → ✅ File accepted
If BOTH fail → ❌ Show error: "Invalid file type!"
