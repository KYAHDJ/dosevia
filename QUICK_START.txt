## TROUBLESHOOTING

**Sound not playing?**
- Check logs: `adb logcat | grep -E "AlarmService|ERROR"`
- Verify file: `adb shell ls /data/data/com.dosevia.app/files/notification_sounds/`
- Try different audio file

**Still using default alarm?**
- Check Settings shows custom sound (not "Default")
- Re-save the sound in Settings
- Check logs for error messages
