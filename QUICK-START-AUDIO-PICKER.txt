   cd dosevia-notifications-SOUND-FIXED


2. Install dependencies (if needed):
   
   npm install


3. Build and sync:
   
   npm run build
   npx cap sync android


4. Run on Android:
   
   npx cap run android

📱 HOW IT WORKS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

User Flow:
1. Go to Settings → Custom Notification Sound
2. Tap "Browse" button
3. File picker opens showing ALL files on phone ✨
4. User navigates to any folder (Music, Downloads, etc.)
5. User selects file

If AUDIO file (✅):
→ File is accepted
→ Waveform appears
→ User can trim and save

If NON-AUDIO file (❌):
→ Error alert appears: "Invalid file type!"
→ File is rejected
→ User can try again

🔒 VALIDATION LOGIC
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

The app checks TWO things:

1. MIME Type: Does it start with "audio/"?
2. File Extension: Is it .mp3, .wav, .m4a, etc.?

If EITHER check passes → File accepted ✅
If BOTH checks fail → File rejected ❌
