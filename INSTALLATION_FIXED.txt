# 🚀 INSTALLATION - REACT DEPENDENCY FIXED

## ✅ WHAT WAS FIXED
The build error was caused by React and react-dom being in `peerDependencies` instead of `dependencies`.

**Fixed:** Moved `react` and `react-dom` to `dependencies` so they install automatically.

---

## 📦 INSTALL & RUN (COPY & PASTE)

```powershell
# 1. Clean install
Remove-Item -Recurse -Force node_modules -ErrorAction SilentlyContinue
Remove-Item -Force package-lock.json -ErrorAction SilentlyContinue
npm install --legacy-peer-deps

# 2. Build and run
npm run build
npx cap sync android
npx cap run android
```

## 📱 DEPLOYMENT ERROR?

The error `ERR_UNKNOWN: Non-zero exit code from adb: 1` usually means:
- ADB connection issue
- Try unplugging and replugging your phone
- Or run: `adb kill-server && adb start-server`
- Then run: `npx cap run android` again
